SystemImager

SystemImager is a software package designed to make it easy for system administrators to install and update large numbers of like machines running Linux.

The current version is 0.24beta which was released on March 10, 2000:
  • systemimager-0.24beta.tar.gz

The last non-beta version is 0.23:
  • systemimager-0.23.tar.gz

  • Older versions are available at the SystemImager download area on SourceForge.

    Please subscribe to the SystemImager mailing list so you can be informed of updates to the software. To subscribe to the SystemImager mailing list, click here: systemimager-request@systemimager.org. Just type the word "subscribe" in the Subject: line. You will recieve a reply confirming your subscription and can then send messages to the list at systemimager@systemimager.org.
