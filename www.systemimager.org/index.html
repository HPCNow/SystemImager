VA SystemImager

VA SystemImager is software that makes the installation of Linux to masses of similar machines relatively easy.

It also makes software distribution, configuration, and operating system updates easy. You can even update from one Linux release version to another! VA SystemImager can also be used for content management on web servers. It is most useful in environments where you have large numbers of identical machines. Some typical environments include: Internet server farms, high performance clusters, computer labs, or corporate desktop environments where all workstations have the same basic hardware configuration.

The current version is 1.1 which was released on May 26, 2000.

See the CHANGE.LOG to find out what has been fixed, broken, discarded, or added!

For RPM based systems:

  • Install this on the imageserver: va-systemimager-1.1-1.i386.rpm
  • Install this on the master client: va-systemimager-client-1.1-1.i386.rpm
  • Here is the source RPM: va-systemimager-1.1-1.src.rpm

Or if you prefer to install from a tar ball (use these for debian systems):

  • Here is the server tar ball: va-systemimager-1.1.tar.gz
  • Here is the client tar ball: va-systemimager-client-1.1.tar.gz

And here is an autoinstallcd image:

  • Here is an ISO image of the autoinstallcd if you prefer not to make it yourself: autoinstallcd-1.0.iso

Older versions are available at the VA SystemImager download area on SourceForge.

You may also need to install the following required packages:
  • nasm-0.98-1.i386.rpm
  • syslinux-1.48-2.i386.rpm
  • tftp-hpa-0.13-1.i386.rpm
  • rsync-2.4.3-1.i386.rpm

Please subscribe to the VA SystemImager mailing list so you can be informed of updates to the software. To subscribe to the VA SystemImager mailing list, click here: systemimager-request@systemimager.org. Just type the word "subscribe" in the Subject: line. You will recieve a reply confirming your subscription and can then send messages to the list at systemimager@systemimager.org.
